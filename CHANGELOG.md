## v1.3.0 (08/06/2026)

### 👥 ניהול משתמשים מלא במערכת
- **חדש:** טאב "משתמשים" בסיידבר (גלוי רק למנהלים)
- **חדש:** טבלת ניהול משתמשים מלאה עם:
  - שם משתמש, שם מלא, אימייל, תפקיד, סטטוס פעיל/מושבת
  - תאריך כניסה אחרונה
  - Badge ויזואלי לכל תפקיד (👑 מנהל / 👤 משתמש)
  - סימון "אני" ליד המשתמש הנוכחי
- **חדש:** הוספת משתמש חדש דרך מודל:
  - שם משתמש (validation: a-z, 0-9, _)
  - שם מלא, אימייל, סיסמה (8 תווים מינימום)
  - בחירת תפקיד: מנהל / משתמש רגיל
- **חדש:** עריכת משתמש קיים:
  - עדכון פרטים (שם מלא, אימייל, תפקיד)
  - שינוי סיסמה אופציונלי (השאר ריק = ללא שינוי)
  - הפעלה/השבתה של החשבון
- **חדש:** מחיקת משתמש עם הגנות:
  - לא ניתן למחוק את עצמך
  - לא ניתן למחוק את המנהל האחרון במערכת
  - אזהרה מיוחדת במחיקת מנהל

### 🛡️ אבטחת המערכת
- כל פעולה דורשת הרשאת `admin`
- סיסמאות מוצפנות bcrypt
- לוג מלא של כל פעולה (`logActivity`)
- שם משתמש לא ניתן לשינוי אחרי יצירה
- משתמש לא פעיל לא יכול להתחבר

### 🔄 איך לעדכן מ-v1.2.x
**רק 2 קבצים להעלאה:**
- `index.html` - עם הטאב והממשק החדש
- `api/users.php` - עם הגנת מנהל אחרון

אין צורך בעדכון DB - הטבלה `users` כבר קיימת.

## v1.2.2 (08/06/2026)

### 🔐 שחזור סיסמה
- **חדש:** קישור "שכחתי סיסמה" במסך הלוגין
- **חדש:** מודל הסבר אינטראקטיבי עם 2 אופציות שחזור (פנייה למנהל / שחזור חירום)
- **חדש:** סקריפט `reset-password.php` לשחזור חירום:
  - הצגת רשימת משתמשים קיימים במערכת (עוזר להיזכר בשם המשתמש)
  - טופס איפוס סיסמה עם אימות (8 תווים מינימום, אישור כפול)
  - לוג של כל ניסיון איפוס ב-`logs/password-reset.log`
  - אזהרת אבטחה אוטומטית אם הקובץ קיים מעל 24 שעות
  - הוראות ברורות למחיקת הקובץ אחרי שימוש

### 🛡️ אבטחה
- הסקריפט מוגן: דורש קיום `config.php` ולא ניתן לשימוש מרחוק ללא גישת FTP
- לוג מלא של כל ניסיון איפוס (משתמש, IP, תאריך)
- ESC סוגר את המודל
- אזהרה ויזואלית בולטת לאחר השימוש

### 🔄 איך להשתמש בשחזור חירום
1. העלה את `reset-password.php` ל-`public_html/report/` דרך FTP/cPanel
2. גש ל-`https://report.bidernet.co.il/reset-password.php`
3. ראה את רשימת המשתמשים → העתק את שם המשתמש שלך
4. הגדר סיסמה חדשה
5. **חובה:** מחק את הקובץ אחרי השימוש!

## v1.0.0 (08/06/2026) - הגרסה הראשונה 🎉

### ✨ תכונות
- מערכת התחברות עם סיסמאות מוצפנות (bcrypt)
- ניהול לקוחות מלא עם לוגו
- דוחות קמפיינים (Facebook + Instagram)
- חיבור Facebook Graph API דרך השרת (פותר CORS)
- סנכרון אוטומטי של קמפיינים
- ייצוא PDF ממותג
- ניהול משתמשים (admin/user)
- יומן פעילות (activity log)
- HTTPS + Security Headers
- מערכת התקנה אוטומטית (install.php)

### 🏗️ ארכיטקטורה
- **Frontend:** HTML/CSS/JavaScript (Single Page)
- **Backend:** PHP 7.4+
- **Database:** MySQL 5.7+
- **Auth:** PHP Sessions
- **Encryption:** AES-256-CBC (Tokens), bcrypt (Passwords)

### 📁 מבנה
```
api/
  ├── config.php          (הגדרות)
  ├── db.php              (חיבור DB)
  ├── helpers.php         (פונקציות עזר)
  ├── auth.php            (התחברות)
  ├── clients.php         (לקוחות)
  ├── campaigns.php       (דוחות)
  ├── users.php           (משתמשים)
  └── facebook-proxy.php  (מתווך FB API)
sql/schema.sql            (סכמת DB)
index.html                (המערכת)
login.html                (התחברות)
install.php               (התקנה)
server-check.php          (בדיקת תאימות)
```
