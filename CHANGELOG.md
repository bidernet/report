## v1.2.1 (08/06/2026)

### 🎬 תצוגת וידאו מלאה במערכת
- **חדש:** כרטיס KPI "צפיות בוידאו" בדשבורד הראשי (מופיע רק כשיש נתוני וידאו)
- **חדש:** סקציית "ביצועי וידאו" מלאה בפרופיל לקוח עם:
  - 4 KPIs גדולים: צפיות, צפו 30+ שניות, סיימו (100%), זמן צפייה ממוצע
  - משפך השלמת צפייה ויזואלי (25% → 50% → 75% → 100%)
  - גרדיאנטים בצבעי כחול-סגול לעיצוב מקצועי
- **חדש:** פיצול וידאו לפי פלטפורמה - כרטיס FB ו-IG מציגים גם צפיות וידאו
- **חדש:** אייקון Video ייעודי (play triangle outline) - מותאם לסגנון line-style
- הצגה חכמה: הכרטיסים מופיעים **רק** כשיש נתוני וידאו (אחרת מוסתרים)
- אחוז סיום וידאו (Completion Rate) מחושב אוטומטית

### 🔧 שדרוגים טכניים
- כרטיס וידאו מופיע במקום הנכון ב-flow של הרינדור (אחרי KPI ולפני הגרפים)
- חישוב ממוצע זמן צפייה מתוחכם - מתעלם מקמפיינים ללא וידאו
- אייקון `icon-video` חדש בסיפרייט עם viewBox מותאם 430x430

### 🔄 איך לעדכן מ-v1.2.0
**פשוט מאוד:** החלף רק את `index.html` בשרת.
- אין צורך בעדכון DB
- אין צורך לסנכרן מחדש (הנתונים כבר ב-DB)
- רענן את הדפדפן (Ctrl+Shift+R)

## v1.2.0 (08/06/2026)

### 🎬 נתוני וידאו
- **חדש:** מערכת תומכת במלוא נתוני הוידאו של Meta API
- 7 שדות וידאו חדשים בקמפיינים: צפיות, p25/p50/p75/p100, צפיית 30+ שניות, זמן צפייה ממוצע
- סנכרון אוטומטי של נתוני וידאו עם כל סנכרון מ-Facebook API
- פיצול נתוני וידאו לפי FB/IG (`fb_video_views`, `ig_video_views`)
- אופציה ל-KPI של "צפיות בוידאו" במודל ה-PDF

### 📅 פילטר תקופה בדשבורד
- **חדש:** 5 כפתורי פילטר תאריך בדשבורד הראשי
- 7 ימים / 14 ימים / 30 ימים / 3 חודשים / כל התקופה
- סינון אוטומטי של KPIs, גרפים ו-PDF
- מידע דינמי על מספר הקמפיינים בתקופה
- ברירת מחדל: כל התקופה

### 🎨 עיצוב נקי - אייקונים line-style
- 14 אייקונים חדשים בסגנון line/stroke מקצועי
- מערכת sprite SVG ל-UI + פונקציית `iconHtml()` ל-PDF
- אייקונים: reach, leads, refresh, budget, engagement, impressions, clicks, ctr, conversions, chart, add, edit, trash, pdf
- כל האייקונים monochrome ויורשים את צבע הטקסט (currentColor)
- החלפת כל האימוג'ים בכפתורי פעולה

### 🖼️ Favicon
- אייקון R צהוב על ירוק כהה בכרטיסיית הדפדפן
- `theme-color` ירוק כהה למובייל
- תמיכה ב-Apple Touch Icon

### 🔧 שדרוגים טכניים
- `facebook-proxy.php` משדרג את `fields` הקריאה ל-Meta API לכולל וידאו
- `campaigns.php` מחזיר int casting לכל שדות הוידאו
- Migration SQL מותאם לעדכון DB קיים (`migration-v1.2.0.sql`)
- הכנה תשתיתית ל-v1.3.0 (Page Insights) - שדות בלקוח + טבלת `page_insights`

### 🔄 איך לעדכן מ-v1.1.0
1. **הרץ את ה-Migration:** עלה ל-phpMyAdmin, הרץ `sql/migration-v1.2.0.sql`
2. **העלה את הקבצים** החדשים (`index.html`, `login.html`, `favicon.jpg`, `api/`)
3. **בצע סנכרון מ-Facebook** - נתוני הוידאו ייטענו אוטומטית
4. אין צורך לחדש Token

### 🔮 מתוכנן ל-v1.3.0 (לא בגרסה הזו)
- UI לבחירת Page בטופס לקוח
- סקציית "נתוני דף עסקי" בפרופיל לקוח
- סנכרון Page Insights (עוקבים, חשיפות אורגניות, מעורבות אורגנית)
- הצגת וידאו בכרטיסי KPI ו-PDF

## v1.0.0 (08/06/2026) - הגרסה הראשונה 🎉

### ✨ תכונות
- מערכת התחברות עם סיסמאות מוצפנות (bcrypt)
- ניהול לקוחות מלא עם לוגו
- דוחות קמפיינים (Facebook + Instagram)
- חיבור Facebook Graph API דרך השרת (פותר CORS)
- סנכרון אוטומטי של קמפיינים
- ייצוא PDF ממותג
- ניהול משתמשים (admin/user)
- יומן פעילות (activity log)
- HTTPS + Security Headers
- מערכת התקנה אוטומטית (install.php)

### 🏗️ ארכיטקטורה
- **Frontend:** HTML/CSS/JavaScript (Single Page)
- **Backend:** PHP 7.4+
- **Database:** MySQL 5.7+
- **Auth:** PHP Sessions
- **Encryption:** AES-256-CBC (Tokens), bcrypt (Passwords)

### 📁 מבנה
```
api/
  ├── config.php          (הגדרות)
  ├── db.php              (חיבור DB)
  ├── helpers.php         (פונקציות עזר)
  ├── auth.php            (התחברות)
  ├── clients.php         (לקוחות)
  ├── campaigns.php       (דוחות)
  ├── users.php           (משתמשים)
  └── facebook-proxy.php  (מתווך FB API)
sql/schema.sql            (סכמת DB)
index.html                (המערכת)
login.html                (התחברות)
install.php               (התקנה)
server-check.php          (בדיקת תאימות)
```
