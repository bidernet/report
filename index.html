איך משיגים את הנתונים?

מדריך מהיר להשגת Ad Account ID ו-Access Token

📌 שלב 1: Ad Account ID 1. היכנס ל-business.facebook.com
2. עבור ל-Ads Manager
3. בראש העמוד תראה את ה-Account ID (מתחיל ב-act_)
4. העתק את כל המספר כולל ה-act_
🔑 שלב 2: Access Token 1. היכנס ל-developers.facebook.com/tools/explorer
2. בחר את האפליקציה bidernet clients
3. לחץ "Generate Access Token"
4. אשר הרשאות: ads_read, ads_management (לקריאה בלבד)
5. העתק את ה-token והדבק כאן
⚠️ Token תקף ל-60 יום ויצטרך חידוש
🔒 אבטחה: ה-Token נשמר רק בדפדפן שלך (localStorage) ולא נשלח לאף שרת חיצוני. רק קריאות ישירות ל-Facebook Graph API.
